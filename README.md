📘 Trabalho Prático – DGT2817

Disciplina: Lógica, Algoritmos e Programação de Computadores

Este projeto reúne as microatividades e o trabalho final da disciplina DGT2817, desenvolvidos em Python. O objetivo é praticar e aplicar conceitos fundamentais de lógica de programação e estruturas básicas da linguagem.

🎯 Objetivos

Utilizar estruturas condicionais if, elif e else.

Aplicar laços de repetição while e for.

Declarar e utilizar funções em Python.

Trabalhar com funções que recebem parâmetros.

Refatorar uma calculadora com boas práticas, funções e estruturas de decisão/repetição.

🛠️ Estrutura do Projeto

Microatividade 1: uso de if/else.

Microatividade 2: uso de if/elif/else.

Microatividade 3: laço while.

Microatividade 4: laço for.

Microatividade 5: função sem parâmetros.

Microatividade 6: função com parâmetros.

Calculadora V2: versão final, integrando os conceitos anteriores.

🚀 Tecnologias

Linguagem: Python 3.x

IDE: VS Code

📌 Conclusão

O trabalho consolidou a prática de estruturas de decisão, repetição e funções, reforçando a lógica de programação aplicada em Python.
